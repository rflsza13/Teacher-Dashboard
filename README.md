# Teacher Dashboard

## Sobre o projeto

Plataforma web para gestão acadêmica de professores, centralizando turmas, alunos, notas, materiais e análise de desempenho em um único sistema.

---

## Objetivo

Criar um ambiente único para organização de aulas e análise de desempenho de turmas, substituindo ferramentas dispersas como planilhas, Drive e anotações externas.

---

## Funcionalidades

### MVP

* Login de professor
* Dashboard principal
* Gestão de turmas
* Gestão de alunos
* Gestão de disciplinas
* Lançamento de notas
* Materiais por aula
* Analytics de desempenho

---

### Futuro

* Controle de frequência
* Integração com Google Drive
* Integração com Google Calendar
* Geração de provas com IA
* Relatórios PDF
